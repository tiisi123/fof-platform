# FOF平台部署脚本说明

### 主部署脚本

| 脚本名称 | 适用场景 | 特点 | 推荐度 |
|---------|---------|------|--------|
| **简化部署-v2.sh** | 生产环境首次部署 | 最新版本，解决所有已知问题 | ⭐⭐⭐⭐⭐ |
| 一键部署.sh | 完整部署（含Git） | 包含Git拉取、备份等完整流程 | ⭐⭐⭐⭐ |

## 推荐部署流程

### 方案1：快速部署（推荐）

适用于大多数场景，使用最新的简化部署脚本。

```bash
cd /root/web/fof/deploy

# 给脚本执行权限
chmod +x 简化部署-v2.sh

# 运行部署
bash 简化部署-v2.sh
```

**特点：**
- ✅ 自动使用 conda 预编译包（避免 GCC 编译问题）
- ✅ 自动修复 numpy/pandas 兼容性
- ✅ 自动跳过 OpenBB（避免依赖冲突）
- ✅ 完整的错误处理和日志输出
- ✅ 预计时间：10-15分钟

---

### 方案2：完整部署（含Git）

适用于需要从Git拉取最新代码的场景。

```bash
cd /root/web/fof/deploy

# 1. 可选：预安装系统依赖
bash 0-预安装依赖.sh

# 2. 运行完整部署
bash 一键部署.sh
```

**特点：**
- ✅ 包含Git拉取和备份
- ✅ 完整的部署流程
- ✅ 适合持续更新场景
- ✅ 预计时间：15-20分钟

---

### 方案3：不安装量化库

适用于不需要量化分析功能的场景。

```bash
cd /root/web/fof/deploy

# 编辑脚本，设置 INSTALL_QUANT_LIBS="no"
vim 简化部署-v2.sh

# 运行部署
bash 简化部署-v2.sh
```

**特点：**
- ✅ 部署速度最快
- ✅ 避免所有编译问题
- ✅ 核心功能完全可用
- ✅ 预计时间：5-8分钟

## 版本对比

### 简化部署-v2.sh（最新版）

**解决的问题：**
1. ✅ GCC 4.8.5 编译失败 → 使用 conda 预编译包
2. ✅ numpy/pandas 二进制不兼容 → 自动重新安装兼容版本
3. ✅ OpenBB 依赖冲突 → 自动跳过安装
4. ✅ 错误处理不完善 → 增强错误提示和交互

**新增功能：**
- 🆕 智能依赖安装（conda + pip 混合）
- 🆕 自动兼容性修复
- 🆕 详细的安装验证
- 🆕 友好的错误提示

**配置选项：**
```bash
INSTALL_QUANT_LIBS="yes"  # 是否安装量化库
```

## 故障排查

### 问题1：后端启动失败（numpy错误）

```bash
# 运行修复脚本
bash deploy/修复numpy冲突.sh

# 选择方案1（推荐）
```

**错误特征：**
```
ValueError: numpy.dtype size changed
```

---

### 问题2：量化库安装失败

```bash
# 运行修复脚本
bash deploy/修复量化库安装.sh

# 选择方案1（conda预编译）
```

**错误特征：**
```
ERROR: Problem encountered: OpenBLAS or Netlib BLAS/CBLAS is required
```

---

### 问题3：OpenBB依赖冲突

```bash
# 卸载OpenBB
bash deploy/清理OpenBB.sh
```

**错误特征：**
```
openbb-core requires fastapi>=0.128.0, but you have fastapi 0.104.1
```

---

### 问题4：服务无法启动

```bash
# 查看详细日志
journalctl -u fof-backend -n 50

# 检查端口占用
netstat -tlnp | grep 8506

# 手动启动测试
cd /root/web/fof/backend
source /root/anaconda3/etc/profile.d/conda.sh
conda activate py310fof
uvicorn app.main:app --host 0.0.0.0 --port 8506
```

## 环境要求

### 必需
- ✅ CentOS 7+ / Ubuntu 18.04+
- ✅ Python 3.10 (conda环境)
- ✅ Node.js 16+
- ✅ Root权限

- 🔧 GCC 4.8+ (或使用conda预编译包)
