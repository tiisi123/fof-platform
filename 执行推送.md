## 🔧 故障排查

### 问题 1: 推送失败 "rejected"

**解决**:
```bash
# 确保使用 --force 参数
git push -u origin main --force
```

### 问题 2: 认证失败

**Gitee**:
```bash
# 使用 SSH
git remote set-url origin git@gitee.com:ts_tiisi/fof_v2.git
git push -u origin main --force
```

**GitHub**:
```bash
# 使用 SSH
git remote set-url github git@github.com:tiisi123/fof-platform.git
git push -u github main --force
```

### 问题 3: 文件太大

如果提示文件太大，检查：
```bash
# 查看大文件
find . -type f -size +50M

# 如果有大文件，添加到 .gitignore
echo "backend/fof.db" >> .gitignore
git add .gitignore
git commit --amend --no-edit
git push -u origin main --force
```
