## 🎯 核心功能

### 流式输出工作流程

```
用户输入问题
    ↓
前端发送请求 (stream: true)
    ↓
后端生成流式响应 (SSE)
    ↓
前端实时接收并显示
    ↓
打字机效果呈现
```

### API 调用示例

```javascript
// 流式模式
POST /api/v1/copilot/chat
{
  "question": "你的问题",
  "stream": true
}

// 响应 (text/event-stream)
data: {"content": "文本片段"}
data: {"done": true}
```
