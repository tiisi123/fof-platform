## 一、产品概述

### 1.1 产品定位

FOF管理平台V2.0是面向**投资经理**的专业级FOF运营管理系统，核心解决：

- **组合管理**：3个FOF组合、约30只子基金的全生命周期管理
- **绩效分析**：从净值分析到持仓穿透的多层次归因分析
- **风险监控**：业绩预警、管理人舆情监控、风险预算管理
- **智能报告**：AI驱动的组合报告、尽调报告自动生成
- **数据集成**：私募排排网API + 邮件爬虫 + Excel导入

### 1.2 目标用户

| 角色 | 人数 | 核心需求 | 使用频率 |
|-----|------|---------|---------|
| **投资经理** | 5人 | 组合分析、调仓决策、报告生成 | 每日 |

## 四、第二期功能详细设计（第2-3个月）

### 4.1 绩效分析深化

#### 4.1.1 Brinson归因（后2层）

**第三层：管理人选择归因**

```
管理人选择效应 = Σ Wp,k × (Rp,k - Rb,k)

其中：
- Wp,k: 组合中管理人k的权重
- Rp,k: 管理人k的实际收益
- Rb,k: 管理人k所属策略的基准收益
```

**第四层：个券选择归因（需四级估值表）**

```
个券选择效应 = Σ Wp,l × (Rp,l - Rb,l)

基于四级估值表的持仓明细，计算：
- 股票选择收益
- 行业配置收益
- 交易时机收益
```

#### 4.1.2 多因子模型（Barra）

**风格因子暴露分析**

| 因子 | 说明 | 计算方式 |
|-----|------|---------|
| 市值因子 | 大小盘暴露 | ln(市值) |
| Beta因子 | 市场敏感度 | 回归系数 |
| 动量因子 | 趋势跟随 | 过去12月收益 |
| 波动因子 | 波动率暴露 | 历史波动率 |
| 价值因子 | 估值水平 | BP/EP |
| 成长因子 | 成长性 | 盈利增速 |
| 流动性因子 | 流动性 | 换手率 |
| 杠杆因子 | 财务杠杆 | 资产负债率 |

**因子归因公式**

```
产品收益 = α + Σ(βi × Fi) + ε

其中：
- α: 特质收益（选股能力）
- βi: 因子i的暴露度
- Fi: 因子i的收益
- ε: 残差
```

#### 4.1.3 四级估值表分析

**数据结构**


### 4.2 AI舆情分析

#### 4.2.1 功能概述

自动爬取并分析管理人及核心人员的公开信息，进行情感分析和事件提取，实现风险预警。

#### 4.2.2 数据来源

| 来源 | 内容 | 频率 |
|-----|------|------|
| 财经新闻 | 东方财富、新浪财经、证券时报 | 每小时 |
| 监管公告 | 中基协、证监会处罚公告 | 每日 |
| 招聘信息 | 猎聘、Boss直聘 | 每周 |
| 社交媒体 | 微博、雪球 | 每日 |

#### 4.2.3 分析能力

**情感分析**
- 正面：获奖、规模增长、业绩优秀
- 中性：人员变动、产品发行
- 负面：处罚、清盘、踩雷、离职

**事件提取**
- 核心人员变动
- 产品清盘
- 监管处罚
- 规模变化
- 投资踩雷

#### 4.2.4 预警推送

**关键词配置**

